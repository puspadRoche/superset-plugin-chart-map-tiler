# superset-plugin-chart-map-tiler
A custom visualization plugin for Apache Superset 3.1.0 that renders a MapLibre GL map using a MapTiler style.json. Includes a fake-data fallback.
See docs/getting-started.md for full steps.
> This repo is **source-only**. Build and run the plugin inside a Superset 3.1.0 frontend environment where `@superset-ui/*` peer deps are present.
> See `docs/getting-started.md` for wiring and usage.
